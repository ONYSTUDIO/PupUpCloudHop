# Game Concept — 안 떨어질개: 구름섬 점프

## 게임명

- **한글명**: 안 떨어질개: 구름섬 점프
- **영문명**: Pup Up! Cloud Hop
- **약칭**: 안떨어질개

## 한 줄 설명

하늘을 떠도는 구름섬을 원터치로 점프하며 강아지와 함께 하늘 높이 올라가는 모바일 캐주얼 게임

## 세계관

구름으로 이루어진 작은 섬들이 하늘 위에 떠 있다.  
각 구름섬은 상승기류나 바람 소용돌이를 중심으로 정해진 궤도를 천천히 돌고 있다.  
강아지 주인공은 이 구름섬들을 오르고 또 올라, 하늘의 끝이 어디인지 확인하려 한다.

## 핵심 플레이

1. 구름섬이 궤도를 따라 움직인다
2. 화면을 터치하면 강아지가 가장 가까운 위쪽 구름섬 방향으로 점프한다
3. 착지에 성공하면 해당 구름섬에 올라탄다 → 점수 +1
4. 그 구름섬도 계속 움직이므로, 타이밍을 맞춰 다음 점프를 준비한다
5. 놓치면 추락 → 게임 오버

## 목표 플랫폼

| 단계 | 플랫폼 |
|------|--------|
| 1차 | 모바일 웹 HTML5 |
| 2차 | Android / iOS 앱 |
| 최종 | Google Play Store / Apple App Store |

## 예상 플레이 시간

- 회차당 1~3분 (캐주얼 세션)
- 최고 기록 경신이 리플레이 동기 부여

## 타깃 유저

- 10대~40대 모바일 유저
- 짧은 시간에 즐길 수 있는 캐주얼 게임을 선호하는 유저
- 귀여운 강아지 테마를 좋아하는 유저

## 디자인 방향

- **원터치 캐주얼**: 복잡한 컨트롤 없이 화면 탭 하나로 플레이
- **직관적 피드백**: 착지 성공 / 실패가 즉각적으로 느껴져야 함
- **귀엽고 아기자기한** 비주얼 스타일
- **무한 도전**: 끝이 없는 구조로 최고 점수 경쟁 유도
